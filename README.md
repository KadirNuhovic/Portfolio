# Kadir Nuhovic - Portfolio

Personal portfolio website built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## 🚀 Features

-   **Multi-page Application** with Next.js App Router
-   **Responsive Design** with Tailwind CSS
-   **Smooth Animations** with Framer Motion
-   **Custom Color Scheme** with shimmer effects
-   **TypeScript** for type safety
-   **Modern UI/UX** with Roboto font

## 📁 Project Structure

```
src/
├── app/                 # Next.js pages and API routes
│   ├── about/page.tsx   # About page
│   ├── skills/page.tsx  # Skills page
│   ├── projects/page.tsx # Projects page
│   ├── contact/page.tsx # Contact page
│   └── api/contact/     # Contact API route
├── components/          # Reusable React components
│   ├── layout/         # Navbar, Footer, Container
│   ├── sections/       # Page sections
│   ├── ui/             # UI components
│   └── animations/     # Animation components
├── data/               # Static data (projects, skills)
├── lib/                # Utilities and constants
└── types/              # TypeScript type definitions
```

## 🛠️ Tech Stack

-   **Framework**: Next.js 16
-   **Language**: TypeScript
-   **Styling**: Tailwind CSS
-   **Animations**: Framer Motion
-   **Icons**: React Icons
-   **Deployment**: Vercel

## 🎨 Design

-   **Colors**: Custom palette with gold accents (#b8a163)
-   **Font**: Roboto
-   **Animations**: Shimmering text effects and smooth transitions
-   **Layout**: Responsive grid system

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.
